FeatureMapper¶

FeatureMapper coordinates the presentation of input patterns, collating and analyzing the responses to measure feature maps, tuning curves, receptive fields, PSTHs, etc.

Introduction¶

To obtain FeatureMapper, run:

git clone git://github.com/ioam/featuremapper.git

Installation¶

FeatureMapper requires Param, ImaGen, and holoviews.
