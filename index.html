FeatureMapper

FeatureMapper coordinates the presentation of input patterns, collating and analyzing the responses to measure feature maps, tuning curves, receptive fields, PSTHs, etc. FeatureMapper was previously part of the Topographica github repository, but is now an independent project that is suitable for use with any simulator or even with experimental data. We are gradually improving the documentation and interface to make it clearer how to use it in this general way.

Installation

FeatureMapper requires NumPy, PIL, ImaGen, and Param.
